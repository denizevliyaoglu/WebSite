WEB TASARIM VE PROGRAMLAMA NEDİR?

Bu diller; sunucu taraflı olmak üzere başlıca, php, .net ve jsp. Tarayıcı/kullanıcı olmak üzere javascript’tir.
Web tasarımı kısmında html ve css de bulunmaktadır.

Html, css, javascript ve php dilleri ile ilgili makalelere, sayfamızın "Makaleler" kısmından ulaşabilirsiniz.
